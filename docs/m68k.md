# M68k Instruction Set Architecture (ISA) Documentation

The simplified Motorola 68000 ISA is a register-based instruction set inspired by the Motorola 68000 architecture. This documentation provides an overview of the instructions available, their syntax, and their semantics.

Comments in Motorola 68000 assembly code are denoted by the `;` character.

## ISA Specific State Views

- `D<n>:dec`, `D<n>:hex` -- Data registers (D0-D7).
- `A<n>:dec`, `A<n>:hex` -- Address registers (A0-A7).
- `PC:dec`, `PC:hex` -- Program Counter.
- `SR:bin` -- Status Register (flags: X, N, Z, V, C).

### Registers

- **Data Registers (D0-D7):** General-purpose registers for arithmetic, logical, and data manipulation.
- **Address Registers (A0-A7):** Used for addressing memory. A7 is also the stack pointer (SP).
- **Program Counter (PC):** Points to the next instruction to execute.
- **Status Register (SR):** Contains condition codes (flags) and system control bits:
  - **X (Extend):** Used for extended precision arithmetic.
  - **N (Negative):** Set if the result of an operation is negative.
  - **Z (Zero):** Set if the result of an operation is zero.
  - **V (Overflow):** Set if an arithmetic overflow occurs.
  - **C (Carry):** Set if a carry or borrow occurs in arithmetic operations.

## Instructions

Instruction size: Variable (16-bit base instruction, with optional extensions for operands).

### Data Movement Instructions

- **Move**
    - **Syntax:** `move <source>, <destination>`
    - **Description:** Move data from the source to the destination.
    - **Operation:** `<destination> <- <source>`
    - **Examples:**
      ```assembly
      move D0, D1       ; Copy D0 to D1
      move #42, D0      ; Load immediate value 42 into D0
      move (A0), D0     ; Load value from memory address in A0 into D0
      move D0, (A1)     ; Store value from D0 into memory address in A1
      ```

- **Load Effective Address**
    - **Syntax:** `lea <source>, <destination>`
    - **Description:** Load the effective address of the source into the destination address register.
    - **Operation:** `<destination> <- address(<source>)`
    - **Examples:**
      ```assembly
      lea (A0, D0), A1  ; Load effective address of A0 + D0 into A1
      ```

- **Exchange**
    - **Syntax:** `exg <register1>, <register2>`
    - **Description:** Exchange the values of two registers.
    - **Operation:** `<register1> <-> <register2>`
    - **Examples:**
      ```assembly
      exg D0, D1        ; Swap values of D0 and D1
      ```

### Arithmetic Instructions

- **Add**
    - **Syntax:** `add <source>, <destination>`
    - **Description:** Add the source to the destination.
    - **Operation:** `<destination> <- <destination> + <source>`
    - **Examples:**
      ```assembly
      add D0, D1        ; D1 <- D1 + D0
      add #10, D0       ; D0 <- D0 + 10
      ```

- **Subtract**
    - **Syntax:** `sub <source>, <destination>`
    - **Description:** Subtract the source from the destination.
    - **Operation:** `<destination> <- <destination> - <source>`
    - **Examples:**
      ```assembly
      sub D0, D1        ; D1 <- D1 - D0
      sub #5, D0        ; D0 <- D0 - 5
      ```

- **Multiply**
    - **Syntax:** `mul <source>, <destination>`
    - **Description:** Multiply the source and destination.
    - **Operation:** `<destination> <- <destination> * <source>`
    - **Examples:**
      ```assembly
      mul D0, D1        ; D1 <- D1 * D0
      ```

- **Divide**
    - **Syntax:** `div <source>, <destination>`
    - **Description:** Divide the destination by the source.
    - **Operation:** `<destination> <- <destination> / <source>`
    - **Examples:**
      ```assembly
      div D0, D1        ; D1 <- D1 / D0
      ```

- **Negate**
    - **Syntax:** `neg <destination>`
    - **Description:** Negate the value in the destination.
    - **Operation:** `<destination> <- -<destination>`
    - **Examples:**
      ```assembly
      neg D0            ; D0 <- -D0
      ```

---

### Logical and Bitwise Instructions

- **And**
    - **Syntax:** `and <source>, <destination>`
    - **Description:** Perform a bitwise AND between the source and destination.
    - **Operation:** `<destination> <- <destination> & <source>`
    - **Examples:**
      ```assembly
      and D0, D1        ; D1 <- D1 & D0
      ```

- **Or**
    - **Syntax:** `or <source>, <destination>`
    - **Description:** Perform a bitwise OR between the source and destination.
    - **Operation:** `<destination> <- <destination> | <source>`
    - **Examples:**
      ```assembly
      or D0, D1         ; D1 <- D1 | D0
      ```

- **Exclusive Or**
    - **Syntax:** `eor <source>, <destination>`
    - **Description:** Perform a bitwise XOR between the source and destination.
    - **Operation:** `<destination> <- <destination> ^ <source>`
    - **Examples:**
      ```assembly
      eor D0, D1        ; D1 <- D1 ^ D0
      ```

- **Shift Left**
    - **Syntax:** `lsl <count>, <destination>`
    - **Description:** Shift the destination left by the specified count.
    - **Operation:** `<destination> <- <destination> << <count>`
    - **Examples:**
      ```assembly
      lsl #1, D0        ; D0 <- D0 << 1
      ```

- **Shift Right**
    - **Syntax:** `lsr <count>, <destination>`
    - **Description:** Shift the destination right by the specified count.
    - **Operation:** `<destination> <- <destination> >> <count>`
    - **Examples:**
      ```assembly
      lsr #1, D0        ; D0 <- D0 >> 1
      ```

---

### Control Flow Instructions

- **Branch**
    - **Syntax:** `bra <label>`
    - **Description:** Unconditional branch to the specified label.
    - **Operation:** `PC <- <label>`
    - **Examples:**
      ```assembly
      bra loop          ; Jump to label "loop"
      ```

- **Branch if Zero**
    - **Syntax:** `beq <label>`
    - **Description:** Branch to the specified label if the zero flag (Z) is set.
    - **Operation:** `if Z == 1 then PC <- <label>`
    - **Examples:**
      ```assembly
      beq done          ; Jump to "done" if Z flag is set
      ```

- **Branch if Not Zero**
    - **Syntax:** `bne <label>`
    - **Description:** Branch to the specified label if the zero flag (Z) is clear.
    - **Operation:** `if Z == 0 then PC <- <label>`
    - **Examples:**
      ```assembly
      bne loop          ; Jump to "loop" if Z flag is clear
      ```

- **Jump**
    - **Syntax:** `jmp <address>`
    - **Description:** Jump to the specified address.
    - **Operation:** `PC <- <address>`
    - **Examples:**
      ```assembly
      jmp (A0)          ; Jump to address in A0
      ```

- **Halt**
    - **Syntax:** `halt`
    - **Description:** Halt the machine.
    - **Operation:** `Stop execution`
    - **Examples:**
      ```assembly
      halt              ; Stop the program
      ```

---

### Stack Instructions

- **Push**
    - **Syntax:** `push <source>`
    - **Description:** Push the source value onto the stack.
    - **Operation:** `SP <- SP - 2; mem[SP] <- <source>`
    - **Examples:**
      ```assembly
      push D0           ; Push D0 onto the stack
      ```

- **Pop**
    - **Syntax:** `pop <destination>`
    - **Description:** Pop the top value from the stack into the destination.
    - **Operation:** `<destination> <- mem[SP]; SP <- SP + 2`
    - **Examples:**
      ```assembly
      pop D0            ; Pop the top of the stack into D0
      ```
